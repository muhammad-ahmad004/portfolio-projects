# Fatima Store - eCommerce Website

Fatima Store is a responsive eCommerce website project developed for a women’s fashion brand. The main goal of this project was to create a clean, modern, and user-friendly online store where customers can easily explore products, view collections, and shop through a smooth digital experience.

## Live Website

https://fatimastore.online/

## Project Type

eCommerce Website / Online Fashion Store

## My Role

Website Design & Development

## Project Overview

This project was focused on building a professional online store for women’s fashion products. The website is designed to present products in a clean and organized way, making it easy for visitors to browse collections, view product details, and navigate through the store.

The design approach was kept simple, responsive, and product-focused so that the website works smoothly across desktop, tablet, and mobile devices.

## Key Features

- Modern eCommerce website layout
- Clean and professional homepage design
- Product-focused store structure
- Product listing pages
- Product detail pages
- Mobile, tablet, and desktop responsive design
